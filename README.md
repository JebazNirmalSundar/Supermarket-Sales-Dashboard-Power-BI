# Supermarket Sales Dashboard (Power BI Project)

## Project Overview

This project presents an interactive Supermarket Sales Dashboard developed using Power BI to analyze sales performance and generate business insights. The dashboard helps track revenue trends, product performance, and customer payment behavior for better decision-making.

## Tools Used

- Power BI
- Excel (Data Preparation)

## Key Features

- Total Sales and Profit analysis
- Monthly and daily sales trend visualization
- Product-wise performance tracking
- Category-wise sales comparison
- Payment mode analysis (Cash vs Online)
- Interactive filters for dynamic reporting

## Business Insights Generated

- Identified top-performing product categories
- Analyzed peak sales months and daily performance trends
- Compared payment preferences across customers
- Supported inventory and sales strategy planning
